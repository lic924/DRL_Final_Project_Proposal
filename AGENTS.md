# Agent Guidance

本儲存庫是 Deep Reinforcement Learning 課程的 proposal-first submission。

## 專案意圖

本專題提出一個 RL buyer negotiation agent，並在 used-car price negotiation environment 中與 LLM-simulated seller opponents 訓練。此儲存庫應誠實呈現目前狀態：它是 final project proposal 與 specification scaffold，不是完成版實作。

## 編輯準則

- 文件應保持正式、精簡且 evidence-oriented。
- 不加入虛構引用、假的實驗結果，或未確認的 Antigravity installation commands。
- 將 OpenSpec files 視為 implementation planning artifacts。
- 保留老師要求的檔案：`start.sh`、`end.sh`、`start.shell`、`end.shell`。
- Shell scripts 應簡單、安全，並避免 destructive checks。
