# OpenSpec 專案

## 專案名稱

使用 LLM 模擬多樣化對手的強化學習議價代理人

## 目的

為 Deep Reinforcement Learning 期末專題準備提案與實作規格。本系統研究 RL buyer agent 是否能透過與 diverse LLM-simulated seller opponents 訓練，改善 negotiation generalization。

## 目前階段

提案與規格階段。此儲存庫不宣稱已完成完整實作。

## 主要能力

規劃中的系統能力是一個 used-car price negotiation environment，包含：

- RL buyer agent。
- LLM-simulated seller。
- controllable opponent personas。
- Q-learning baseline implementation。
- seen and unseen opponent evaluation。
